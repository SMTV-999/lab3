«Блоги» - хранится информация о пользователях (логин, ФИО) и их записях (заголовок, текст, автор, дата). Приложение должно позволять:
1. Добавлять информацию о блогерах и новых записях.
2. Изменять содержимое записи.
3. Удалять выбранную запись и весь блог целиком (удаление учетной записи пользователя со всеми записями).
4. Выводить все записи во всех блогах за выбранную дату.
5. Выводить список авторов, отсортированный по количеству записей в базе, написанных данным автором.
